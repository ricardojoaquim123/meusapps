Política de Privacidade

CarLife Log — versão web

Última atualização: 06/01/2026

Visão geral

O CarLife Log prioriza a privacidade: a maior parte dos dados gerados pelo usuário é armazenada localmente no dispositivo e só é transferida quando o usuário solicita exportação ou backup.

Dados que tratamos

Coletamos e armazenamos localmente informações do veículo, histórico de manutenção, fotos anexadas e registros de abastecimento. Não coletamos dados extras automaticamente nem vendemos informações pessoais.

Anúncios e identificadores

Se exibirmos anúncios (por exemplo via AdMob), os provedores podem usar identificadores de publicidade e dados técnicos para veiculação. Para transparência, publicamos app-ads.txt na raiz do site.

Compartilhamento com terceiros

Compartilhamos dados apenas quando o usuário autoriza (exportação, backup ou integração com serviços externos). Provedores de anúncios podem receber dados técnicos conforme suas próprias políticas.

Segurança

Adotamos medidas para proteger dados no dispositivo e durante exportações. Recomendamos manter o app atualizado e backups seguros. Nenhuma transmissão via internet é feita sem consentimento explícito do usuário.

Contato

Para dúvidas sobre privacidade envie um email para: suporte@milapps.example (substitua pelo contato oficial).

Alterações nesta política

Atualizamos esta política ocasionalmente. A versão mais recente sempre estará disponível nesta página.
