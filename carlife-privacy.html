Política de Privacidade

CarLife Log — versão web

1. Introdução

Esta Política de Privacidade descreve como o CarLife Log coleta, usa, compartilha e protege informações dos usuários. Nosso foco é manter a privacidade e a segurança dos dados, especialmente porque o app é projetado para operar localmente (offline) sempre que possível.

2. Dados coletados

O CarLife Log armazena dados localmente no dispositivo do usuário, incluindo: informações do veículo, registros de manutenção, fotos anexadas e histórico de abastecimentos. O aplicativo não envia esses dados a servidores remotos, exceto quando o usuário opta por realizar backup ou exportar relatórios.

3. Uso de anúncios e identificadores

Se o app utilizar serviços de anúncios (por exemplo, AdMob), os provedores de anúncios podem coletar identificadores de publicidade e dados técnicos necessários para veicular anúncios. Para transparência, mantemos o arquivo app-ads.txt publicado na raiz deste site para verificação de parceiros de ads.

4. Compartilhamento e terceiros

Compartilhamos dados com terceiros somente quando o usuário realiza uma exportação, backup ou quando integrações externas (por exemplo: serviços de armazenamento que o usuário escolher) forem explicitamente autorizadas. Nós também podemos compartilhar informações técnicas mínimas com provedores de serviços de anúncios conforme suas políticas.

5. Segurança

Adotamos práticas para proteger os dados localmente no dispositivo e durante exportações. No entanto, nenhum método é 100% infalível; recomendamos que o usuário mantenha backups seguros e atualize o app quando novas versões estiverem disponíveis.

6. Contato

Para dúvidas relacionadas à privacidade, entre em contato via email: suporte@milapps.example (substitua pelo contato oficial).

7. Atualizações desta política

Podemos atualizar esta política periodicamente. Publicaremos a versão mais recente nesta página e indicaremos a data da última atualização.
